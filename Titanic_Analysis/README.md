# 📖 Overview

The **Titanic Survival Analysis** project is an Exploratory Data Analysis (EDA) application built with Python. It analyzes the famous Titanic passenger dataset to uncover survival trends using data cleaning, statistical exploration, and visualizations.

The project demonstrates a complete beginner-friendly data analysis workflow, including loading data, handling missing values, exploring dataset information, and generating insightful charts with Matplotlib and Seaborn.

---

# 🎯 Objectives

- Load and inspect the Titanic dataset
- Understand dataset structure
- Detect and clean missing values
- Explore passenger information
- Analyze survival patterns
- Visualize relationships between different features
- Learn the complete EDA workflow

- Number of rows
- Number of columns
- Data types

## 🧹 Data Cleaning

Cleaning operations include:

- Fill missing Age values using Median
- Fill missing Embarked values using Mode
- Remove Cabin column because of excessive missing data

# 📦 Installation

Clone Repository

```bash
git clone https://github.com/yourusername/Titanic-Survival-Analysis.git
```

Move into project folder

```bash
cd Titanic-Survival-Analysis
```

Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

Run the project

```bash
python titanic_analysis.py
```
