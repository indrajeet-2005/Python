# 📖 Overview

**Pandas Data Analyzer & Data Visualization** is a Python-based command-line application designed to simplify the process of loading, exploring, cleaning, analyzing, and visualizing CSV datasets.

Built using **Pandas** and **Matplotlib**, this project demonstrates the complete basic data analysis workflow—from importing raw data to generating meaningful visual insights through different chart types.

It is an excellent project for students and beginners learning **Data Science**, **Data Analysis**, and **Python Libraries**.

# 📦 Installation

Clone Repository

```bash
git clone https://github.com/yourusername/Pandas-Data-Analyzer.git
```

Move into Project

```bash
cd Pandas-Data-Analyzer
```

Install Dependencies

```bash
pip install pandas matplotlib
```

Run Application

```bash
python "Pandas Analyzer & Data Visualization.py"
```
