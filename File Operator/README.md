# 📚 About The Project

**Personal Journal Manager** is a command-line application developed in Python that allows users to maintain a digital journal. Users can write daily notes, view all previous entries, search for specific entries using keywords, and delete all journal records when needed.

The application stores all journal entries in a simple text file (`journal.txt`), making it lightweight, fast, and easy to understand for beginners learning Python File Handling.
