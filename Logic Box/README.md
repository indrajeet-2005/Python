# 📖 About The Project

**Logic Box** is a beginner-friendly Python command-line application designed to strengthen programming logic and problem-solving skills.

The application provides two interactive modules:

- ⭐ Pattern Generator
- 🔢 Number Analyzer

Through these modules, users can practice loops, conditional statements, arithmetic operations, and menu-driven programming while gaining confidence in Python fundamentals.

This project is an excellent starting point for students preparing for coding interviews, Python assignments, and programming challenges.

# ✨ Features

## ⭐ Pattern Generator

Generate a right-angled star (`*`) pattern by entering the desired number of rows.

Example:

```
Input:

5

Output:

*

**

***

****

*****
```

---

## 🔢 Number Analyzer

Analyze a range of numbers entered by the user.

The program automatically:

- Identifies Even Numbers
- Identifies Odd Numbers
- Calculates the Total Sum
- Displays results in sequence

Example

```
Range

1 - 5

Output

1 → Odd

2 → Even

3 → Odd

4 → Even

5 → Odd

Sum = 15
```

---

## 📋 Menu Driven Interface

Interactive command-line menu:

```
1. Generate Pattern

2. Analyze Numbers

3. Exit
```
