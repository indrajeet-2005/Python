# 📖 About The Project

**Data Analyzer & Transformer** is a Python command-line application that demonstrates the practical implementation of **Functional Programming** concepts while performing basic data analysis tasks.

The application allows users to input numerical datasets and perform various analytical operations such as statistical summaries, sorting, filtering, and factorial calculations through an interactive menu-driven interface.

This project is ideal for beginners who want to strengthen their understanding of Python functions, recursion, lambda expressions, built-in functions, and data processing.

# ✨ Features

## 📥 Input Dataset

Users can enter multiple numbers separated by spaces.

Example

```
10 25 18 45 60 90
```

---

## 📊 Display Data Summary

Shows:

- Total Elements
- Minimum Value
- Maximum Value
- Sum
- Average

---

## 🧮 Factorial Calculator

Calculates factorial using **Recursion**.

Example

```
Input : 5

Output : 120
```

---

## 🔍 Filter Dataset

Filter numbers greater than or equal to a user-defined threshold using **Lambda** and **Filter**.

Example

Threshold

```
50
```

Output

```
[60, 90]
```

---

## 📈 Sort Dataset

Supports:

- Ascending Order
- Descending Order

Uses Python's built-in **sorted()** function.

---

## 📋 Dataset Statistics

Displays:

- Minimum
- Maximum
- Sum
- Average
