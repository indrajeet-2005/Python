# 📖 About Project

Managing personal finances is one of the most important aspects of everyday life. Many people struggle to understand where their money is being spent.

The **Smart Expense Tracker** is a Python-based financial management system designed to solve this problem by allowing users to:

- Store expenses
- Categorize transactions
- Generate financial reports
- Analyze spending habits
- Visualize data using charts

This project demonstrates practical implementation of Python programming, Object-Oriented Programming, Data Analysis, and Data Visualization.

# 🚀 Features

## 📝 Add Expense

Users can store

- Date
- Amount
- Category
- Description

Data automatically gets stored inside

```
expenses.csv
```

---

## 📊 Expense Summary

Displays

- Total Spending
- Average Spending
- Category-wise Spending

Uses

- Pandas
- NumPy

---

## 🔍 Filter Expenses

Search expenses according to category.

Example

```
Food
Travel
Shopping
Bills
Entertainment
Medical
Education
```

---

## 📅 Monthly Report

Automatically groups expenses month-wise.

Example

```
January : ₹8500

February : ₹7600

March : ₹9200
```

### 📊 Bar Chart

---

### 📈 Line Chart

### 🥧 Pie Chart

# 💻 Installation

Clone Repository

```bash
git clone https://github.com/YourUsername/Smart-Expense-Tracker.git
```

Open Folder

```bash
cd Smart-Expense-Tracker
```

Install Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

Run

```bash
python expense_tracker.py
```
