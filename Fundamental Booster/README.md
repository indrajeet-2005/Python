# рџ“– About The Project

**Fundamental Booster** is a beginner-friendly Python project designed to help learners understand the core concepts of Python programming through an interactive command-line application.

The program collects basic personal information from the user, displays the stored values along with their data types and memory addresses, and performs a simple calculation to estimate the user's birth year.

This project is ideal for students who are starting their Python programming journey and want hands-on experience with fundamental programming concepts.
