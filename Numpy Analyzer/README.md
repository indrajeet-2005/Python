# 📖 Overview

**NumPy Analyzer** is a command-line application developed using **Python** and **NumPy** to demonstrate the power of numerical computing through an interactive, menu-driven interface.

The project allows users to create one-dimensional, two-dimensional, and three-dimensional arrays, perform mathematical operations, analyze statistics, combine arrays, apply indexing and slicing, search and sort data, and explore common NumPy concepts in a structured way.

The code is organized using **Object-Oriented Programming (OOP)**, making it clean, reusable, and easy to extend.

# 🚀 Functional Modules

## 📦 Array Creation

Supports

- 1D Arrays
- 2D Arrays
- 3D Arrays

Users can dynamically create arrays of different dimensions by providing input values.

---

## ✂️ Indexing & Slicing

For 2D arrays the toolkit supports

- Element Indexing
- Matrix Slicing
- Row Selection
- Column Selection

Perfect for understanding image-like matrix operations.

---

## 🔢 Mathematical Operations

Supports

- Addition
- Subtraction
- Multiplication
- Division

Element-wise operations are performed between arrays having identical dimensions.

---

## 🔗 Array Combination

Supports vertical stacking of multiple arrays.

Useful for

- Dataset merging
- Image matrix operations
- Numerical data processing

---

## 🔍 Search • Sort • Filter

Search

- Locate values using NumPy indexing

Sort

- Arrange array values

Filter

- Display values greater than a user-defined threshold

---

## 📊 Statistical Analysis

Compute

- Sum
- Mean
- Median
- Standard Deviation
- Variance

Ideal for quick data analysis.

# 🛠 Installation

Clone Repository

```bash
git clone https://github.com/yourusername/Numpy-Analyzer.git
```

Move into project

```bash
cd Numpy-Analyzer
```

Install dependency

```bash
pip install numpy
```

Run Project

```bash
python "Numpy Analyzer.py"
```
